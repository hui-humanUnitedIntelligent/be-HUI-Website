# Archive

Dieser Ordner enthält alte oder ungenutzte Dateien aus dem HUI-Webseitenprojekt,
die für zukünftige Referenz aufbewahrt werden.

## Inhalt

- `logo` – Alte Logo-Datei (PNG ohne Dateiendung), ersetzt durch `logo.png`

## Hinweis

Diese Dateien werden von keiner aktiven Seite mehr verwendet.
Sie können nach einer Überprüfungsphase gelöscht werden.
